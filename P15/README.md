Repositori ini memuat dokumentasi laporan dan draf terkait penelitian **Vulnerability Assessment** yang diimplementasikan pada website profil desa menggunakan platform **Helium Security**.

## 📌 Tentang Penelitian
Penelitian ini bertujuan untuk menguji tingkat keamanan infrastruktur website pemerintahan desa dari ancaman siber (seperti SQL Injection, XSS, dll.) menggunakan metode *ethical hacking* dalam bentuk pemindaian celah keamanan (*vulnerability assessment*) tanpa merusak stabilitas sistem publik.

* **Objektif:** Mengidentifikasi celah keamanan secara praktis dan efektif sebagai tindakan preventif.

## 🛠️ Instrumen & Metodologi

### Alat & Bahan
* **Platform Utama:** Helium Security (Fitur: *Website Scanner*)
* **Konfigurasi Pengujian:** *External Scan* dengan opsi *Basic Scan*
* **Perangkat Pendukung:** Perangkat laptop, koneksi jaringan internet, dan peramban (*web browser*).

### Alur Penelitian (Methodology)
```text
Mulai ➔ Buat Akun Helium ➔ Login Dashboard ➔ Daftarkan Target URL ➔ Pilih External Basic Scan ➔ Jalankan Website Scanner ➔ Proses Scanning ➔ Analisis Hasil ➔ Kesimpulan ➔ Selesai
