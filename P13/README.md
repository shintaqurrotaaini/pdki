# P13 - Pembuatan Dokumen Standar Keamanan Informasi & Contoh SOP

## Deskripsi
Folder ini berisi tugas mata kuliah **Keamanan Informasi** mengenai penyusunan **Dokumen Standar Keamanan Informasi** dan **Standar Operasional Prosedur (SOP)** dengan studi kasus **Pengelolaan Folder Tugas Kuliah**.

## Isi Folder

- **Dokumen_Standar_Keamanan_Informasi_Folder_Tugas_Kuliah.pdf**  
  Berisi dokumen standar keamanan informasi, SOP pengelolaan folder tugas kuliah, analisis risiko, dan penutup.

## Tujuan

- Menyusun dokumen standar keamanan informasi.
- Menentukan kebijakan keamanan dalam pengelolaan dokumen akademik.
- Menyusun SOP penyimpanan, backup, restore, dan berbagi dokumen.
- Menerapkan prinsip **Confidentiality**, **Integrity**, dan **Availability (CIA Triad)** dalam pengelolaan informasi.
